Actividad Hola Mundo

Objetivo
Practicar cuatro formas básicas de visualizar un mensaje en JavaScript:
- alerta emergente
- mensaje en el documento
- salida por consola
- mensaje dentro de un div

Cómo ejecutar
Descargar los archivos y abrir actividad_hola_mundo.html en el navegador.

Evidencias
Se incluyen capturas de:
- alerta emergente
- mensaje en página
- salida por consola
- mensaje dentro del div

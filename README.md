Actividad Hola Mundo

Objetivo
Practicar cuatro formas básicas de visualizar un mensaje en JavaScript:
- alerta emergente
- mensaje en el documento
- salida por consola
- mensaje dentro de un div

Cómo ejecutar
 ir al link `http://localhost/ActividadSemana5/actividad_hola_mundo.html` en el navegador.


Evidencias
Se incluyen capturas de:
- alerta emergente
